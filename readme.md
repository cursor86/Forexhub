# Global Currency Converter & Forex Education Hub
### Live Site: [https://github.io](https://github.io)

## Project Overview
This project is an all-in-one financial utility developed by **SS Traders Education**. It combines a real-time, high-accuracy currency conversion tool with a comprehensive library of educational resources designed to improve global financial literacy.

## Key Features
- **Live Mid-Market Rates**: Real-time conversion for over 150 global currencies using secure API integration.
- **Dynamic Market Snapshots**: A live-updating table showcasing popular trading pairs (USD/AUD, USD/EUR, etc.).
- **Educational Hub**: Contains over 20 expert-led articles and a 500-word deep-dive guide on finding the best money exchange centres.
- **SEO & Adsense Optimized**: Built with semantic HTML5, valid schema structures, and mandatory legal disclosures (Privacy Policy & Terms) for Google AdSense compliance.

## Technical Stack
- **Frontend**: Clean HTML5 and CSS3 (Responsive Mobile-First Design).
- **Logic**: Vanilla JavaScript utilizing the Fetch API for real-time data retrieval.
- **API**: Integration with ExchangeRate-API for accurate mid-market values.

## Educational Goals
The primary mission of this repository is to empower travellers, students, and traders by providing them with the "real" exchange rate—bypassing the hidden fees typically found at banks and airports.

## Compliance
- **Privacy Policy**: Includes disclosures for the Google DoubleClick DART cookie.
- **Accessibility**: ARIA labels and semantic structure for screen-reader compatibility.
- **Robots.txt & Sitemap**: Configured for optimal search engine crawling.
